## ğŸ”§ Troubleshooting Print Issues

### Issue: Print button doesn't work
**Solutions:**
- Try keyboard shortcut (Ctrl+P or Cmd+P)
- Use browser menu â†’ Print
- Check if pop-ups are blocked
- Refresh the page

### Issue: Print layout looks wrong
**Solutions:**
- Check print preview before printing
- Ensure scale is set to 100%
- Try different browser
- Update browser to latest version

### Issue: Logo doesn't print
**Solutions:**
- Enable "Background graphics" in print settings
- Use "Print to PDF" then print the PDF
- Check printer settings

### Issue: Text is cut off
**Solutions:**
- Reduce margins in print settings
- Check paper size matches print size
- Try "Fit to page" option
- Use smaller margin (0.5cm)
